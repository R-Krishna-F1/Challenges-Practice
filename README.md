# Challenges / Practice

This repository contains my writeups and notes from various Android security challenges and CTF competitions. The work here covers mobile application security — reverse engineering Android APKs, hooking native functions using Frida, decompiling bytecode with jadx, analyzing native libraries with Ghidra, and bypassing security checks in real challenge apps. Platforms include HackTheBox, Frida Labs, Payatu BugBazaar, Hex Tree IO, and more.

All writeups are in the [`docs/`](./docs) folder, automatically synced from my Notion workspace and kept up to date via GitHub Actions.
